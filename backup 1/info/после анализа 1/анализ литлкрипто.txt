Небольшой лайфхак для бота

Чтобы не дёргать API по 100 раз в секунду:

Логируй wallet-balance по сегментам в локальную БД (например, SQLite/PostgreSQL).

Ставь TTL кэша (5–10 сек на балансы), а рыночные данные — в отдельный канал подписок (WS) с собственным кэшем/агрегацией.

Если хочешь, дам готовый набор из трёх Postman-запросов на wallet-balance (UNIFIED/SPOT/CONTRACT) с уже подставленным coin=USDT, чтобы просто кликать по очереди.
+ когда будем работать с маиннет, нужно напоминание чтобы ключу маиннета права выдать
